## sejarah singkat 

dalam sejarahnya singkatnya bahasa ini di ciptakan dari bahasa C yang sebelumnya untuk di permudah, C++ di kembangkan oleh Bjarne Stroustrup dan teamnya di Bell Laboratories (AT&T, USA) dengan tujuan simulasi implementasi OOP dan dalam ke efisienan, kemudian 1989 ANSI (american Nasional Standard Institute) menstandarisasi penggunaan bahasa C++ tujuannnya untuk mendapatkan sebanyak mungkin orang atau vendor yang dapat menggunakan bahasa pemograman C++, di 1998 ISO (International Organization for Standarization) mengakui standar untuk bahasa pemograman C++ (ISO/IEC 14882)

## karasteristik C++
C++ bukalah murni OOP melainkan hybrida yang memiliki fungsi dari program C, artinya anda memiliki fitur yang ada dalam C ;
> -universally <br> -efficient,to the machine learning <br> -portable programs for various platforms.

kumpulan source dari C yang ada juga dapat di gunakan dalam program C++.
C++ supports pada konsep dari OOP yang mana;
> -data abstraction, that is, creation of clasess to describe objects <br> -data encapsulation for controlled acsess to object data <br> -inheritance by creating derived clasess (including multiple derived classes) <br> -polymorphism (Greek for multiform), that is, the implementation of instructions that can have varying during program execution.
